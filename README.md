# Kannada Sign Language Recognition (KSL)

## 🚀 Project Overview
Real-time sign language recognition using Deep Learning (CNN) and NLP.

## 🧠 Features
- Real-time gesture detection
- Kannada text generation (vattakshara support)
- Speech output (TTS)
- Web-based UI

## 🛠 Tech Stack
- Python, TensorFlow, OpenCV
- FastAPI, WebSockets
- React.js

## ▶️ How to Run
1. Install dependencies
2. Run backend:
   uvicorn main:app --reload
3. Open frontend

## 📊 Results
- Accuracy: ~97%
- Real-time performance (~180 ms latency)
